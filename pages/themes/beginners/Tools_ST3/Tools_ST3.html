Защо избрах Sublime Text?

Мултиплатформен: Linux, Windows, Mac OS X.
Гъвкав, мощен и изключително бърз!
Може да се използва за обучение безплатно, без ограничения във функционалността и времето.
Изчерпателна документация: Sublime Text Unofficial Documentation.
Многобройна общност, създаваща непрекъснато множество полезни добавки за най-новите технологии: Package Control - The Sublime Text package manager.
Python-базиран, с добре разработен API позволяващ да добавяте нова функционалност.

Инсталиране

От официалния сайт, където е последната версия: Sublime Text 3 - Download

Повече информация: ST3 installation notes

Настройки

Глобални настройки

Използват се 2 файла:
Sublime Text 3 => Preferences => Settings – Deafult
Sublime Text 3 => Preferences => Settings – User
Файлът, в който записваме промените е Settings – User.Това гарантира, че настройките ни ще се запазят дори и след ъпдейт на Sublime.

Настройки за проект

Sublime Text 3 => Project => Edit Project.

Структурата на файла е описана тук: Sublime Text 3 Documentation: Projects

Примерни потребителски настройки


            {
                "color_scheme": "Packages/User/Solarized (Light) (SL).tmTheme",
                "font_face": "Courier New",
                "font_size": 11,
                "draw_white_space": "all",
                "rulers": [80],
                "tab_size": 4,
                "translate_tabs_to_spaces": true,
                "theme": "Soda Light 3.sublime­theme",
                "show_encoding": true,
            }

Клавишни комбинации (Keyboard Shortcuts)

Default:
Sublime Text 3 => Preferences => Key Bindings ­ Default
Потребителски:
Sublime Text 3 => Preferences => Key Bindings – User
Детайлна информация: http://sublime-text-unofficial-documentation.readthedocs.org/ en/latest/reference/keyboard_shortcuts_win.html

Пример:

В Sublime Text 3 => Preferences => Key Bindings - User:


        [
            { "keys": ["alt+shift+up"], "command": "select_lines", "args": 
            {"forward": false} },
            { "keys": ["alt+shift+down"], "command": "select_lines", "args": 
            {"forward": true} },
        ]

Въвеждане на команди


The Command Palette


Активиране: Ctrl+Shift+P
Дава бърз достъп до всички поддържани от ST команди
Филтриране/търсене по произволна част от низа

Инсталиране на Package Control

Отворете: https://packagecontrol.io/installation
Копирайте точно съдържанието на блока в сиво, под надписа SUBLIME TEXT 3
Отворете Sublime Text 3
Активирайте конзолата на Sublime, чрез чрез натискане на ctrl+` (Control + BackTick)
Поставете копираното съдържание в конзолата.
Добре е да рестартирате ST (задължително под Windows).
